使用github pages搭个博客

选择博客

对博客的页面自己没有太多的需求，能分类和分标签显示是其中一个比较重要的需求。试了下Jekyll的默认主题minima要支持分类的话需要自动改动下模版。就在自己打算改改模版的时候无意中发现了基于hexo的NexT主题，这个主题很是符合自己对博客的需求。惰性使然，转战hexo。
